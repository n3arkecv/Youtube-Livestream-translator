# Assets 資源資料夾

此資料夾用於存放應用程式的資源檔案：

- `icon.ico` - 應用程式圖標（建議尺寸：256x256）
- `icon.png` - PNG格式圖標
- 其他UI資源檔案

## 添加圖標

要為應用程式添加圖標：

1. 準備一個圖標檔案（建議使用256x256像素的PNG）
2. 使用線上工具或圖像編輯軟體轉換為.ico格式
3. 命名為 `icon.ico` 並放置在此資料夾中
4. 重新執行 `build_exe.bat` 建立帶圖標的執行檔

推薦的圖標轉換工具：
- https://convertio.co/png-ico/
- https://www.icoconverter.com/
